Track My Studies

The Idea Behind the Application

The application

Track My Studies is an interactive web application to track how many hours are being planned and effectively used for specific tasks.

Assuming that there are 4 different modules a student can take, it is up to the user to add to one of the modules a specific activity.

Along with that, the user should insert how much time - in hours - is expected to be spent and what actually was used for that task.

The user can also delete an activity from the list and select multiple activities to calculate the total of hours planned and done.

Technologies

XML - The structure data is stored in XML format, information not available to the client.
XLST - Parsing the XML data into HTML file available to the client.
HTML - Displaying the page content on the web browser.
CSS - Used for some page styling.
BOOTSTRAP - Used for some styling and specific settings as Navigation Bars, Jumbotron and Tables.
Font Awesome and Google Fonts - For font styling and icons.
Node.js - For server data retrieval and manipulation.
JavaScript - Functions and validation.
